# MatsuDex Source

This document is the internal source reference for the CreatureMatsu Dex project.

It defines the official creature index, including creature number, name, pillar, category, Matsu group, and main action/function keyword.

---

## 1. Purpose

**MatsuDex Source** is used as the master creature list for the CreatureMatsu Dex website.

It helps keep the website consistent when adding, checking, sorting, filtering, or documenting creatures.

This file is not the same as the full creature data files used by the website.
It is a source/index document.

---

## 2. Source Role

```txt
MatsuDex.docx = master creature list
MATSUDEX_SOURCE.md = markdown source version for GitHub documentation
DEX_STRUCTURE.md = technical structure of the website
creature-data/ = actual JavaScript data used by the website
```

## 6. Matsu Order Logic

The repeated Matsu order is:

```txt
Oso
Kara
Choro
Ichi
Jyushi
Todo
```

For Origins categories, each category contains six creatures in Matsu order.

For Variation pillars, each Matsu receives six creatures per pillar.
